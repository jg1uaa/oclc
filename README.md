oclc
====

OpenCL offline compiler.


## Usage

```
$ ./oclc kernel.cl
```


## Build

- Clone submodule

```
$ git submodule init
$ git submodule update
```

- Build binary

```
$ make
```


## Test compiled kernel binary

```
$ make test
```
